# StudySmash
CS160 Software Engineering Group Project

## How to run

### Frontend
```
cd frontend
npm install
npm run dev
```

### Backend
```
cd backend
```
Mac: 
```
python3 -m venv venv
source venv/bin/activate
```
Windows: 
```
python -m venv venv
venv\Scripts\activate
```
```
pip install -r requirements.txt
python app.py
```
