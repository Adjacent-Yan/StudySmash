# StudySmash
CS160 Software Engineering Group Project

## How to run

### Frontend
```
cd frontend
npm install
npm run dev
```

### Backend
Mac: 
```
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```
Windows: 
```
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```
